### 기능

```
1. 회원가입, 로그인 등 유저 정보 관련 CRUD
2. 제품 목록과 제품 상세 정보를 조회 가능
3. 장바구니에 넣은 상품을 주문할 수 있음
4. 주문 완료 후 주문 내역 조회/취소가 가능함
```

### 서비스 기획

- 페르소나:
  반려동물용품에 관심이 있는 사람
- 목적:
  반려동물용품을 잘 모르는 사람도 간편하고 쉽게 관련 제품을 구매 할 수 있는 온라인 쇼핑몰 서비스
